## Project Goal

KIT-STARTDY เป็น Starter Kit สำหรับการเรียนรู้และสร้างหุ่นยนต์ ที่ช่วยให้ผู้เริ่มต้นสามารถสร้างโปรเจกต์จริงได้ตั้งแต่วันแรก

---

## Core Flow

ผู้ใช้เลือกชุดหุ่นยนต์ → ศึกษาคู่มือและวงจร → ประกอบหุ่นยนต์ → อัปโหลดโปรแกรม → หุ่นยนต์ทำงานจริง

---

## Sprint 3 Plan

* พัฒนาต้นแบบแขนกลหุ่นยนต์
* ทดสอบการควบคุม Servo ด้วย ESP32
* จัดทำคู่มือการประกอบแบบ Step-by-Step
* จัดเก็บหลักฐานการพัฒนาและการทดสอบ

---

## How to Run / How to Use

1. ประกอบหุ่นยนต์ตามคู่มือ
2. เชื่อมต่อ ESP32 กับคอมพิวเตอร์
3. เปิด Arduino IDE
4. อัปโหลดโค้ดตัวอย่าง
5. ทดสอบการทำงานของหุ่นยนต์
